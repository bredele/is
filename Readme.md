
# truthy

  simple util that return true if truthy

## Installation

    $ component install bredele/truthy

## API

### is#truthy(data)

  return true if truthy

### is#defined(data)

  return false if undefined

## Example
```js
var is = require('is');

is.truthy(true);
is.defined(undefined);
```
